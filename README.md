# TeenStack

Конструктор мини-SaaS для подростков и молодых фрилансеров.

## Как запустить локально

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173

## Как выложить в интернет на Vercel (бесплатно)

### Шаг 1. Залить код на GitHub

1. Зайди на github.com, создай аккаунт если его нет
2. Создай новый репозиторий (New repository), назови `teenstack`, оставь публичным или приватным — не важно
3. Загрузи все файлы из этой папки в репозиторий (можно через "uploading an existing file" в браузере — просто перетащи все файлы)

### Шаг 2. Подключить Vercel

1. Зайди на vercel.com, войди через GitHub
2. Нажми "Add New" → "Project"
3. Выбери репозиторий `teenstack`
4. Vercel сам определит, что это Vite-проект — настройки трогать не нужно
5. Нажми "Deploy"

Через 1-2 минуты сайт будет жить на адресе вида `teenstack.vercel.app` — это уже настоящий публичный сайт в интернете, доступный с любого устройства.

### Шаг 3 (опционально). Свой домен

В настройках проекта на Vercel → Settings → Domains можно подключить купленный домен (например teenstack.ru).

## Важно про Supabase

В файле `src/App.jsx` уже вшиты твои ключи Supabase (URL и anon key). Это нормально — anon key безопасен для публичного использования, если в Supabase включена Row Level Security (она уже включена в нашей схеме).

Если захочешь отключить обязательное подтверждение email при регистрации (чтобы пользователи сразу логинились без письма):
Supabase Dashboard → Authentication → Settings → выключи "Confirm email"

## Про Google Play

Это веб-приложение (сайт). Чтобы попасть в Google Play, есть два пути:

1. **Trusted Web Activity (TWA)** — обернуть готовый сайт в Android-приложение через Bubblewrap или PWABuilder. Это самый быстрый путь, занимает несколько часов, требует уже работающего сайта на собственном домене с HTTPS.
2. **Полный rewrite на React Native / Flutter** — намного дольше (недели), но даёт нативный опыт.

Для начала имеет смысл довести веб-версию до стабильной работы, набрать первых пользователей, а потом уже паковать в Google Play через TWA.
